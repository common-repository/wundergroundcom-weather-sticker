Contributors: dropdeaddick
Tags: weather, widget

Adds a sidebar widget to display the standard wunderground.com Weather Sticker

== Description ==



== Installation ==

1. Upload `wunder.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add to sidebar and configure location throught the 'widgets' menu in WordPress
